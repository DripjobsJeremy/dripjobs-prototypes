Activation Dashboard — All Accounts

CS Team · Super Admin View · All tenant accounts, no date restriction · Updated Mar 31, 2026

What’s new in this view:
All tenants are now visible regardless of signup date. Legacy accounts (created before 12/28/25) show Total Invoiced Sales (lifetime) and — for Days Remaining — no 90-day activation rules apply. Net-new accounts (after 12/28/25) continue tracking the 90-day activation window. New columns added: Industry, Days Remaining, Plan, /Month, and 4 add-on status columns. Plan and /Month read from Company Settings (local). Totals Row updates dynamically with filters.
⚠ Prototype Only — Login Data Not Yet Tracked
The login columns below are simulated for demonstration purposes. As confirmed with TJ (June 16, 2026), DripJobs does not currently capture or store per-user login timestamps. This prototype was created for Mary Murphy (CSM) to demonstrate what this data could look like.  Three event types are modeled: (1) TRUE LOGIN — real tenant user authenticating;  (2) ADMIN ACCESS — CS/Super User accessing tenant account;  (3) IMPERSONATION — CS/Super User impersonating a specific tenant user. The Last Login column always surfaces the most recent True Login. Admin access appears as a secondary sub-label so CS can distinguish stale true-login data from recent admin activity.
✨ New in v1.8 — CS Feedback Flag (Phase 2 Prototype)
The CS Feedback column (pink group) is manually set by the assigned OBCSS team member. It surfaces low satisfaction signals from Intercom CSAT or Onboarding star ratings that cannot be pulled automatically. Click any cell in the CS Feedback column to set flags independently per source. Each account can carry both an Intercom signal and an OB signal simultaneously — both display as stacked badges with source and date.  Intercom CSAT: 😟 Bad (Neutral — does not affect CSAT score)  or  😡 Terrible (Negative — reduces CSAT score).  OB Feedback: ⭐⭐ 2 Stars or ⭐ 1 Star. This column requires a separate ticket from the V1 spec.
