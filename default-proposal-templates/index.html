Welcome to your Proposal Templates!
Proposal Templates are reusable blueprints that speed up every bid — load one when creating a new proposal and your line items, packages, and settings are ready to go.
  • Create a template for each service type you offer (e.g. Interior Painting, Exterior + Pressure Wash) so you're never starting from scratch.
  • Set a default template and it will automatically pre-load whenever you start a new proposal.
  • Add Packages (Good / Better / Best tiers) inside a template to give customers pricing options on a single proposal — proven to increase average ticket size.
  • Click any template name below to edit its line items, packages, payment settings, and more.

How Proposal Templates Work
Proposal templates are reusable starting points that appear in the template dropdown when creating a new proposal — saving you time on every bid.
  • When starting a new proposal, select this template from the dropdown to load it instantly.
  • To price by square footage: Enter the sq ft as the quantity and your price per sq ft — DripJobs calculates the total automatically.
  • Want pre-set pricing? Update the price per unit in Products / Services, then edit this template to pull in those rates — so your template is always ready to send.

How Package Templates Work
Packages let you present Good / Better / Best pricing tiers on a single proposal — giving customers a choice while increasing your average ticket.
  • Each package (Bronze, Silver, Gold) loads as a selectable option when this template is used on a new proposal.
  • Set a fixed package price directly on each package card — this overrides individual line item totals and is what the customer sees.
  • Line items within each package serve as a scope-of-work checklist. Mark items as "Description only" to show what's included without breaking out pricing.
  • Tip: Use the "Recommended" badge on your best-value package to guide customers toward the option that's best for them — and best for your business.
