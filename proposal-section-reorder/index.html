Show About Us Section

Developer Annotations

BR-1 — Eligible Sections Trust Builders, Details, Media, Custom Section(s), About Us are individually moveable with the Move button.
BR-2 — Locked Groups Line Items (Line Items + Optional Line Items) and Areas (Areas + Optional Areas) always move together as a single unit. Packages is a separate moveable row.
BR-3 — Fixed Anchors Hero Image is always position 1; Your Investment / Financing is always position 10; Terms and Conditions is always last. No Move button shown on any of these.
BR-4 — Move Interaction Clicking Move on an eligible row opens an inline placement panel showing "Before [Section]" options for all other moveable positions. Selecting one closes the panel and reorders the list.
BR-5 — Restore Default "Restore Default" button resets all rows to BR-6 order. Shows confirmation toast.
BR-6 — Default Order Hero Image → Trust Builders → Details → Packages → Line Items → Areas → Media → Custom Section(s) → About Us → Your Investment / Financing → Terms and Conditions
BR-7 — Scope Company-level setting. Applied to all proposals for the tenant.
BR-8 — About Us Integration Existing standalone About Us Move control is superseded. About Us is a moveable row here. Toggle below the list controls its visibility (does not remove it from order list).
Edge Case: About Us toggled off Row still appears in position list. Toggle state is separate from position. Section does not render on proposal when disabled.
Out of Scope V1 Drag-and-drop, per-proposal overrides, hiding other sections.
