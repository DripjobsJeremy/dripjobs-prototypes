Messaging & Automations

2 updates

New First Name Keywords for Drips, Reminders & Templates

NewJul 8
What changed
Added two new dynamic keywords — {salesperson-first-name} and {user-first-name} — available anywhere you already use keywords: drip sequence steps, reminders, and message templates.
Why it matters
Write messages that sound like a real person talking, using first names instead of full names.
Action needed
Update your templates if you'd like to start using the new keywords.
